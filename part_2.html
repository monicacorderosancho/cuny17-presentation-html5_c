Introducción a CSS 3

CSS: Conceptos básicos

Cascading Stylesheets (CSS) es la tecnología que usamos para darle diagramación y estilo a nuestras páginas web.

¿Cómo CSS afecta un documento HTML?

Los navegadores aplican las reglas CSS a un documento modificando el modo en el que lo presenta.

Aprendamos cómo se forma una regla CSS

Composición de un reglas CSS

  • Propiedades: Un conjunto de propiedades, que tienen valores, los cuales modifican como el HTML es presentado.

  • Selectores: Un selector, el cual selección uno o varios elementos para aplicar las valores del conjunto de propiedades.
